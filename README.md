# BFSI Regulatory & Finance Control Room — Dashboard Summary

**Live at:** https://akzato.github.io/BFSI-control-dashboard-/

**Built by:** Akshat Kapoor · June 2026 · Open source

### What it is
A single-page interactive dashboard covering Indian banking, insurance, and lending regulations (RBI/IRDAI), paired with a Gen AI layer and practical finance tools. Built as a personal learning/portfolio project — not a professional or official deliverable.

Five tabs

1. Regulatory Domains** (7 cards, filterable)
- Insurance Distribution — one-click sales ban, suitability checks, NBFC distribution rules
- Bank Types & Oversight — risk-based deposit insurance, recovery-agent code
- Fixed Deposits — minimum tenure, premature withdrawal rules, penalty disclosure
- NBFC vs Non-NBFC — Scale-Based Regulation layers (Base/Middle/Upper/Top)
- Home Loans — foreclosure ban on floating rate, LTV caps
- Car Loans — same foreclosure rules, fixed vs floating distinction
- Two-Wheeler Loans — NBFC-heavy channel, disclosure parity
- Toggle switch** reveals a "Gen AI applied" use case under each card (e.g. KFS-compliance checker, complaint-pattern classifier)

2. Tax & Benefits
- New vs old tax regime slab table (FY 2025-26)
- 87A rebate, standard deduction, 80C/80D limits
- Business/self-employed specific notes (regime-switching rules, presumptive tax)
- **Live calculator** comparing tax payable under both regimes

3. Loan Eligibility by City Tier
- Tier 1/2/3 city classification with example cities
- Salary-slab eligibility tables for home, car, and two-wheeler loans

4. Which Loan/Insurance Fits You**
- Input salary, dependents, age, city tier → get directional suggestions for vehicle loan type, term insurance cover, and health insurance cover

5. Methodology
- Research approach and explicit disclaimer that content is AI-assisted research, not professional advice — verify against rbi.org.in / irdai.gov.in / incometax.gov.in

Design
Navy/teal "control room" aesthetic, Georgia serif headers, monospace labels — built to look like a compliance reference tool rather than a generic template.
